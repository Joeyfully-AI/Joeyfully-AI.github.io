# Research

## Selected Research

### Failure-Aware Reflection for Vision-Language-Action Models

**Keywords:** Vision-Language-Action Models, Robot Learning, Physical Consistency, Failure Recovery

This project studies how to improve long-horizon VLA execution through failure-aware reflection and mirror-consistent feasibility evaluation. The goal is to make action selection more physically grounded and robust to execution discrepancies.

**Links:** Paper / Code / Project page coming soon.

---

### RL Token Reproduction: Efficient and Accurate VLA Control via RL Token Representations

**Keywords:** VLA, Reinforcement Learning, SmolVLA, LeRobot, LIBERO

This project is a low-resource RLT-style reproduction for VLA control. It investigates whether latent action-token representations can be used to refine imitation learning policies through reinforcement learning.

**Links:**

- [Model Repository](https://huggingface.co/Joeyfully/smolvla_rlt_libero_10)

---

### Multimodal Robot Assistant System

**Keywords:** LLM, GraphRAG, Knowledge Graph, Multimodal Interaction, Mobile Robot

This project builds a multimodal intelligent assistant system for mobile robots, focusing on experimental teaching scenarios. The system integrates LLM-based interaction, retrieval augmentation, and knowledge graph components.

## Research Interests

I am broadly interested in reliable robot learning. My current focus includes:

- Physically consistent Vision-Language-Action models
- Failure-aware policy improvement
- Reinforcement learning refinement for robot policies
- Long-horizon robotic manipulation
- Embodied reasoning and multimodal robot systems
